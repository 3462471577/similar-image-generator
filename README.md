**Q: 上传原图后，生成的图片和原图会一模一样吗？**  
**A:** 不会。AI 会根据您上传的原图作为“参考底图（垫图）”，提取其构图、色调或轮廓，并结合您输入的“提示词”重新绘制。您可以控制相似度，让它高度还原，或是仅仅借鉴原图的某个特定元素。

**Q: 这种“相似图生成”适合用来做什么？**  
**A:** 非常适合：1. **画师/设计师**：上传草图，让 AI 细化上色；上传一张成图，生成同风格的系列插画。2. **自媒体**：将真实照片转化为动漫头像、像素风插图。3. **电商**：更换商品图的背景环境，保留商品主体特征。

**Q: 如何让 AI 生成的图片更贴近我的预期？**
